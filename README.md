# AI VANGUARD Dashboard

A modern, high-tech dashboard for tracking the latest Artificial Intelligence news. Built with Streamlit and Google News RSS.

## Features
- **Real-time News**: Fetches latest AI news via Google News RSS.
- **Glassmorphism Design**: Premium, dark-themed UI with responsive grid layout.
- **Search & Filters**: Custom search and quick filters for Generative AI, Robotics, etc.
- **Japanese Support**: Optimized for Japanese news sources.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
streamlit run app.py
```
